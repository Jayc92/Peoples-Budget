# Beta Feedback Template (Owner Use)

Copy/paste this per tester report when triaging. Pairs with
`docs/beta-feedback-operations.md` for category/label definitions.

```
Tester:
Device/browser:
Flow completed? yes/no
What confused them:
What broke:
Turnstile reaction:
ZIP/county explanation reaction:
Priority:
Owner notes:
Decision:
- fix now
- fix later
- ignore / not aligned
```

Notes:
- "Priority" — use the triage labels from `docs/beta-feedback-operations.md`
  (`blocker`, `bug`, `ux-confusion`, `mobile`, `privacy-trust`,
  `feature-request`, `later`).
- Keep this file for owner-only triage; do not publish filled-out copies
  containing anything a tester shared that could identify them.
